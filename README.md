# Fraud Detection System

## Project Overview
This project is a Machine Learning based Fraud Detection System developed using the IEEE-CIS Fraud Detection dataset. The aim of this project is to detect fraudulent financial transactions using machine learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- XGBoost
- SMOTE

## Steps Performed
1. Data Loading and Merging
2. Data Preprocessing
3. Missing Value Handling
4. Label Encoding
5. Train-Test Split
6. Data Balancing using SMOTE
7. XGBoost Model Training
8. Model Evaluation

## Model Performance
- Accuracy: 96.6%
- Precision: 40.2%
- Recall: 47.6%
- F1 Score: 43.6%
- AUC-ROC Score: 72.8%

## Conclusion
The model successfully identifies fraudulent transactions using XGBoost and handles imbalanced data using SMOTE.
